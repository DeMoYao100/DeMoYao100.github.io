# file

一个类 lz77 压缩的压缩算法，改变的地方在于滑块长度固定二

具体规则是：建立缓存

每次出现新的双字节（两个 ascii，即滑块长），则在该字节前面加一 bit 的 '1'，并存入缓冲区，如果在缓存找到已有，则在前面加 8bit 的 '0'，然后下面 8bit 做填充和寻址（在缓存内找到相应的双字节）
