  • - qemu 运行过程（虽然题目中是缺 so 的，但是确实是这么个流程）
    1. sudo apt install git libglib2.0-dev libfdt-dev libpixman-1-dev zlib1g-dev
    2. git clone https://git.qemu.org/git/qemu.git
    3. cd qemu
    4. git checkout v5.0.0
    5. ./configure --target-list=tilegx-linux-user
    6. make
    7. cd tilegx-linux-user/
    8. ./qemu-tilegx ./Desktop/newFolder/chall
  • - objdump 配置过程：
    1. wget https://ftp.gnu.org/gnu/binutils/binutils-2.35.tar.gz
    2. tar xvf binutils-2.35.tar.gz
    3. cd binutils-2.35
    4. mkdir build && cd build
    5. ../configure --target=tilegx-linux-gnu
    6. make
    7. tilegx-linux-gnu-objdump -d ~/Desktop/newFolder/chall > output.asm
    8. 即可获得文件的汇编
